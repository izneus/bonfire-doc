# 介绍
Bonfire 是一个基于 Spring Boot 的后端开发框架，主要用于研发中小企业级的前后端分离的信息化项目。
主要的技术选型为 Spring Boot、Spring Security、Mybaits Plus、Redis & Jwt、Swagger。

## 特性
- **开发者友好** Bonfire 的代码简洁易懂，注释丰富，开发者可以轻易理解和修改逻辑。
- **功能强大** 采用主流技术栈，针对企业协同开发优化代码结构和代码规范，轻松提交高质量代码。
- **开箱即用** 直接进行业务，自带丰富的部署脚本，一键创建部署环境，发布应用。

## 项目目标
- **初学者** 让 java 初学者快速入门基于 spring 的主流 java web 开发流程、代码规范，上手开发相关的软件和工具的使用，配合文档以及代码内自带的注释、范例代码，暂时性忽略让人云里雾里的众多技术术语和眼花缭乱的@注解，专注于 Ctrl + C 和 Ctrl + V 出生产级代码，到点下班。
- **项目组** 让正在技术选型的项目组快速构建适合自己的开发框架和开发规范，方便对项目组新人统一培训，快速进行业务开发，降低机会成本。
- **老司机** 欢迎 Pull Request，或给我们报告 Bug。

## 内置功能
- 用户管理：系统用户的配置管理。
- 角色管理：rbac一部分，角色管理以及角色权限分配。
- 权限管理：rbac一部分，权限实体管理。
- 字典管理：系统字典管理。
- 访问日志：自动记录系统运行日志和客户端的api调用日志。
- 任务调度：基于quartz的任务调度以及任务管理。
- 服务监控：提供服务器系统、内存、磁盘、jvm等相关数据。
- 公告消息：系统公告、点对点、点对面的消息发送功能。
- 工单管理：系统工单的创建、回复、关闭和其他管理功能。
- 代码生成：自动生成crud相关各层代码
- 文件管理：统一的上传、下载以及文件管理
- 其他内置功能请查看后端手册
